Installing FreeDMG


Requirements

Operating system requirements:
Mac OS X v.10.4.x
Mac OS X v.10.5.x (10.5.2 required for encryption in 10.5)
Mac OS X v.10.6

Mac OS X v.10.5.2 is required to use encryption in Leopard. Encryption is functional in all versions of 10.4.

Hardware requirements:
Mac OS X supported disc burner required to burn images

Software requirements:
Xcode tools must be installed to create Software License Agreements

Installation

To install FreeDMG, drag the program to your Applications folder.

Updating FreeDMG

FreeDMG does not currently have an auto-update mechanism available. To update the software, download the latest version of FreeDMG from:

http://www.kelleycomputing.net/freedmg

Drag the new version of FreeDMG into your Applications folder, confirming that you'd like to replace the older version.

*An important note on updating software: In most situations, upgrades will maintain preferences. In some situations, old preference files can cause problems when a newer version of the software is used. Although all efforts are taken to make newer versions of FreeDMG backward-compatible with older versions and their preferences, it may be necessary to remove preferences when upgrading to newer versions of a program. If crashes at startup or other problems are encountered, attempt to remove FreeDMG preferences from ~/Library/Preferences/net.kelleycomputing.FreeDMG.plist.
