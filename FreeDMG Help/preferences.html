Format
Image formats

UDRW - UDIF read/write image

A read/write image which can have its contents modified. These images are
limited to the specified size of the image. Resizing read/write images is
possible. Read/write images are (on disk) bit-for-bit as large as their
partition map specifies.

note: SPARSE type images allow for images that grow/shrink with the data.

UDRO - UDIF read-only image

UDIF read-only images are disk images that cannot be modified. Similar to
UDRW disk images (but read-only), UDRO images are the same size on disk as their
partition map specifies.

UDCO - UDIF ADC-compressed image

UDCO disk images are read-only compressed. These images utilize compression
